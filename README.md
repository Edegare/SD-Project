# Distributed Systems Project

## Requirements
- JavaSDK

## Usage
Inside your repository:
1. Compile:
```bash
./gradlew build
```
2. Start server:
```bash
java -cp build/classes/java/main server.Server
```
3. Start client:
```bash
java -cp build/classes/java/main client.Client
```

4. Cleanup:
```bash
./gradlew clean
```

## Installation

```bash
git clone https://github.com/your-username/SD-Project.git
```

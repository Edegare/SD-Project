# Distributed Systems Project

## Requirements
- JavaSDK
- ApacheMaven

## Usage
Inside your repository:
1. Clean and Compile:
```bash
mvn clean compile
```
2. Start server:
```bash
mvn exec:java -Pserver
```
3. Start client:
```bash
mvn exec:java -Pclient
```

0. Just clean:
```bash
mvn clean
```
0. Just compile:
```bash
mvn compile
```

## Installation

```bash
git clone https://github.com/your-username/SD-Project.git
```

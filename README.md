# Distributed Systems Project

## Requirements
- JavaSDK

## Usage
Inside your repository:
1. Compile:
```bash
javac -d out src/*.java
```
2. Start server:
```bash
cd out/
java MainServer
```
3. Start client:
```bash
cd out/
java MainClient
```

## Installation

```bash
git clone https://github.com/your-username/SD-Project.git
```
